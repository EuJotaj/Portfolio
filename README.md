# Desenvolvedor Front-End Portfólio | Janildo Júnior (JOTA)

Este é o repositório do meu portfólio pessoal "JOTA", desenvolvido para demonstrar minhas habilidades técnicas, projetos e experiência profissional como desenvolvedor Front-End.

## 🚀 Tecnologias e Arquitetura

Este projeto foi construído **sem o uso de frameworks**, utilizando a raiz das tecnologias web para garantir a máxima performance e controle total sobre cada detalhe visual e interação no DOM. Todo o estilismo Tailwind foi desconstruído e reescrito de forma limpa.

- **HTML5 Semântico:** Estruturação otimizada para legibilidade e SEO.
- **CSS3 (Vanilla):** Utilização intensiva de Variáveis (Custom Properties), Flexbox, CSS Grid e dezenas de animações otimizadas.
- **JavaScript Puro (Vanilla JS):** Acessos ao DOM, *Intersection Observers* e comportamentos de `scroll`.
- **SVGs Nativos:** Dispensando importações pesadas de bibliotecas de ícones para manipular *fills* e comportamentos diretamente no vetor.

## ✨ Funcionalidades e Destaques Visuais

- **Carrossel Infinito (Tech Stack):** Uma esteira horizontal de ícones SVG com looping semântico sem cortes. Gira infinitamente com CSS Keyframes e pausa em hover.
- **Efeito "Tilt" 3D e Glassmorphism:** Cards de projetos que possuem fundos translúcidos difusos (`backdrop-filter`) e acompanham delicadamente o eixo do mouse com matemática simples.
- **Scroll Spy & Scroll Suave:** A barra de navegação rastreia ativamente em qual seção da página o visualizador está e destaca de forma inteligente a posição.
- **Revelação em Scroll (Fade-In):** Todos os blocos, títulos e containers entram na tela de forma gradual de forma otimizada usando a `IntersectionObserver` API do navegador.
- **100% Responsivo:** O layout se reajusta naturalmente desde um smartphone de tela pequena até displays Ultrawide, mantendo o aspecto moderno.

## 🛠️ Como Executar Localmente

Como não existem empacotadores, *bundlers* (como Webpack/Vite) ou compiladores, rodar este portfólio não exige nenhuma biblioteca de sistema. 

1. Faça o clone deste diretório:
   ```bash
   git clone https://github.com/EuJotaj/Portfolio.git
   ```
2. Abra seu diretório:
   ```bash
   cd Portfolio
   ```
3. Você pode simplesmente dar um clique-duplo no arquivo `portifolio.html` em qualquer navegador. 
   *(Opcional: Para uma experiência de desenvolvedor, abra a pasta no VS Code e rode um **Live Server** para ver atualizações ao vivo).*
